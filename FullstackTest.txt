## Tugas 1

### Membuat Api Login

- Endpoint `/login`
- Method: `POST`
- Expected Request Format

```json
{
    "username": "admin",
    "password": "pastibisa",
}

```

- Expected Response Format

```json
{
    "status": "success / error",
    "message": "pesan sukses / error",
    "data": {
        "token": "token untuk autentikasi",
        "admin": {
            "id": "uuid admin",
            "name": "nama admin",
            "username": "username admin",
            "phone": "no telepon admin",
            "email": "email admin",
        },
    }
}

```
